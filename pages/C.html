Introduction


The C language is a
- universal,
- procedural imperative
computer programming language
developed by
- Dennis M. Ritchie
in
- 1972
at
- Bell Telephone Laboratories
to develop the
- UNIX operating system.
C is the
- most widely used computer language.

The C language
combines elements of
- high-level languages
with
- assembly language
performance and works close to the computer hardware.
It allows you to
- manipulate data at a low level
and at the same time
- use high-level constructs
to
- control the operation of the program.

The C language was and remained
- one of the most widespread programming languages.
This language have found application in many projects,
including both in
- operating systems
and in varius
- application software
for many devices -
from
- supercomputers to embedded systems.
